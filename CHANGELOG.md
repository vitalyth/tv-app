# Changelog

## [0.1.3] - 2026-05-21
### Added
- Recent VOD refresh sources for Keshet 12, Reshet 13, and Now 14 with channel-specific discovery rules.
- Recently added VOD carousel on the VOD page.

### Changed
- Refined VOD navigation, carousel layout, search placement, and recent viewing presentation.
- Split channel-specific recent VOD search logic into dedicated backend source modules.

## [0.1.2] - 2026-05-07
### Added
- Program guide now highlights the currently playing channel, row, and live program.
- Alternate channel sources can be selected from a source menu while duplicate guide rows stay collapsed.
- The current time marker now includes a time label and stays visible above program hover states.

### Changed
- Reworked the guide to use a single scroll container with sticky channel and time headers for smoother synchronized scrolling.
- Improved player overlay behavior in the guide, including drag positioning, mobile drag-handle visibility, and z-index ordering.
- Player position is preserved when changing channels and reset when the player is closed.
